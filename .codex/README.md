# .codexx

Pasta de contexto para continuar o projeto em outro computador ou outra conversa.

Leia nesta ordem:

1. `PROJECT_CONTEXT.md` — visão geral, stack, arquivos e variáveis.
2. `BOLAO_RULES.md` — regras de pontuação, ranking, chibis e jogos simultâneos.
3. `DEPLOY_CHECKLIST.md` — como rodar, subir e manter backup.

Prompt recomendado para uma nova conversa:

```text
Leia a pasta .codexx primeiro para entender o contexto do projeto Bolão da Copa e depois me ajude com a próxima alteração.
```
